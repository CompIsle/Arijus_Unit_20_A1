CSS Example of Inline Linking

This text will have burlywood background colour and green text.
<div class="inlinelink" style="background-color: burlywood; color: #185;">This text will have burlywood background colour and green text.</div>

CSS Example of Header Linking

This type of linking will have the same outcome as Inline Linking.

Code:

<head> <title>Examples of CSS</title> <h1>Examples of margins, padding and borders.</h1> <style> .headerlink { background-color: burlywood; color: #185; } </style> </head> <div class="headerlink">This type of linking will have the same outcome as Inline Linking.</div>

CSS Example of External Linking

This type of linking will also have the same outcome as Inline and Header linking.
<div class="externallink">This type of linking will also have the same outcome as Inline and Header linking.</div> <style> .externallink { background-color:burlywood; color: #185; } </style>

This code will be stored in an external file called "styles.css", it's almost the same as "Header Linking", but is stored outside of the HTML code.
